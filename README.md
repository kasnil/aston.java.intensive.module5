# Командный итоговый проект

В этом проекте используется [Gradle](https://gradle.org/).
Для сборки и запуска приложения используйте окно инструментов *Gradle*, щелкнув значок Gradle на правой панели инструментов,
или запустите его непосредственно из терминала:

* Запустите `./gradlew run` для сборки и запуска приложения.

* Запустите `./gradlew build` для сборки только приложения.

* Запустите `./gradlew check` для выполнения всех проверок, включая тесты.

* Запустите `./gradlew clean` для очистки всех результатов сборки.

Обратите внимание на использование Gradle Wrapper (`./gradlew`).
